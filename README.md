# Personal Autobiography Website

## Overview

This project is a simple Personal Autobiography Website developed using HTML5 and CSS3. It demonstrates the use of semantic HTML elements, responsive web design, and external CSS styling. The website presents personal information, hobbies, and favorite things in a clean and organized layout.

---

## Features

- HTML5 semantic structure
- Responsive design
- Smooth scrolling navigation
- Horizontal navigation bar with hover effects
- Profile section with image and personal information
- Life story section:
  - My Childhood
  - Teenage Life
  - Adulthood
- Hobbies section
- Favorites section with images and descriptions
- Styled header and footer
- Mobile-friendly layout

---

## Technologies Used

- HTML5
- CSS3
